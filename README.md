# Supply Chain Analytics Project

## Overview
This project analyzes supply chain data and builds predictive models to forecast future trends.

## Objectives
- Clean and preprocess raw data
- Perform exploratory data analysis (EDA)
- Build predictive models:
  - Naïve Bayes
  - Holt-Winters time series forecasting

## Tools & Technologies
- Python (pandas, matplotlib, scikit-learn)
- Jupyter Notebook

## Key Results
- Identified key trends in supply chain performance
- Built forecasting models for future demand
